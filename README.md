Проект: Pцех печати на футболках (конструктор на одной странице)
Описание - MVP-страница-конструктор на одной панели: выбор футболки, редактор нанесения и расчёт цены. - Витрина: 5 примеров работ, 5 отзывов. - Страницы/меню внизу: карта/услуги/политика и реквизиты. - Тема: Nexus по умолчанию, светлый/темный режим.
Структура репозитория - design-test.html - styles.css - design-test.md (если понадобится) - README.md - assets/ (моки, изображения)
Названия файлов-советы - design-test.html — основная тестовая страница под конструктор - styles.css — стили для токенов Nexus и базовой верстки - README.md — документация проекта и инструкции по развёртыванию на GitHub Pages
Как скачать и использовать - Скопируйте содержимое каждого блока к соответствующему файлу: design-test.html, styles.css, README.md. - Откройте design-test.html локально в браузере для проверки. - В репозитории создайте ветку main, залейте файлы и включите GitHub Actions/Pages деплой, если нужно.
